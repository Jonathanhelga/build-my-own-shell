// ⏺ Here's each piece explained:                                                                                                                                          
                                                                                                                                                                      
  ---                                                                                                                                                                   
  1. Define the struct — like a blueprint for what each operator "means":                                                                                               
  struct RedirectOp {                                                                                                                                                   
      int fd;      // which output: STDOUT_FILENO (1) or STDERR_FILENO (2)                                                                                            
      bool append; // true = append (>>), false = overwrite (>)                                                                                                         
  };                                                                                                                                                                    
                                                                                                                                                                        
  ---                                                                                                                                                                   
  2. The map — the actual collection storing all operators:                                                                                                             
  const std::map<std::string, RedirectOp> REDIRECT_OPS = {                                                                                                              
      {">",   {STDOUT_FILENO, false}},  // overwrite stdout                                                                                                             
      {"1>",  {STDOUT_FILENO, false}},  // same as >                                                                                                                    
      {"2>",  {STDERR_FILENO, false}},  // overwrite stderr                                                                                                             
      {">>",  {STDOUT_FILENO, true }},  // append stdout                                                                                                                
      {"2>>", {STDERR_FILENO, true }},  // append stderr                                                                                                                
  };                                                                                                                                                                    
  std::map<std::string, RedirectOp> means: key is a string (the operator), value is a RedirectOp struct.                                                                
                                                                                                                                                                        
  ---                                                                                                                                                                   
  3. Check if a token is a redirect operator:                                                                                                                         
  if (REDIRECT_OPS.count(token)) { ... }                                                                                                                                
  .count() returns 1 if the key exists, 0 if not.                                                                                                                     
                                                                                                                                                                        
  ---                                                                                                                                                                   
  4. Get the operator's info:
  RedirectOp op = REDIRECT_OPS.at(token);                                                                                                                               
  // now use:                                                                                                                                                         
  op.fd;      // tells you which file descriptor to redirect                                                                                                            
  op.append;  // tells you O_APPEND or O_TRUNC                                                                                                                          
                                                                                                                                                                        
  ---                                                                                                                                                                   
  So instead of 4 separate booleans, you just do:                                                                                                                       
  // before: is_redirect_exists || is_redirect_error_exists || is_operator_appends_exists ...                                                                           
  // after:                                                                                                                                                             
  if (REDIRECT_OPS.count(token)) { ... }
